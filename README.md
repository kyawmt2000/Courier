# Courier API

FastAPI backend for the courier app.

## Render settings

- Runtime: Python 3
- Build Command: `pip install -r requirements.txt`
- Start Command: `uvicorn main:app --host 0.0.0.0 --port $PORT`

## Test endpoints

- `GET /`
- `POST /auth/login`
- `GET /orders`
- `POST /orders`
- `GET /docs`
